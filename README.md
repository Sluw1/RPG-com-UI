# RPG com UI 🎮

Um jogo RPG feito com HTML, CSS e JavaScript puro, com sistema de exploração, batalhas e progressão de personagem.

## 🚀 Demonstração
https://app.netlify.com/projects/rpgui/

## ⚔️ Funcionalidades
- Sistema de exploração
- Combate por turnos
- Inimigos e bosses
- Sistema de níveis e atributos
- UI interativa

## 🧠 Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)

## 🎯 Objetivo
Projeto desenvolvido para praticar lógica de programação, manipulação do DOM e construção de jogos em JavaScript puro.
